# Examples

Getting-started examples for the GitHub Copilot SDK.

## Available Examples

| Example | Description | Path |
|---------|-------------|------|
| **Hello World** | Minimal Copilot SDK agent | [hello-world/nodejs/](hello-world/nodejs/) |
| **Weather Assistant** | Weather assistant with tool calling | [weather-assistant/nodejs/](weather-assistant/nodejs/) |

## Running an Example

```bash
# Hello World
cd hello-world/nodejs && npm install && npx tsx hello-world.ts

# Weather Assistant
cd weather-assistant/nodejs && npm install && npx tsx weather-assistant.ts
```
