# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - Initial Release

### Added
- AKS Monitor Agent (Node.js) - cluster health monitoring with GitHub issue creation
- Multi-language support: Node.js, Python, Go, .NET starter templates
- CI/CD pipeline templates: GitHub Actions, Azure DevOps, GitLab CI, Bitbucket, CircleCI
- CLI runner script for local use case execution
- MCP server integration (GitHub, Azure, AKS)
- SDK utilities per language
- Getting started examples (Hello World, Weather Assistant)
- Comprehensive documentation
